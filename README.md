# Real-Time Chat Application

A real-time chat application built to demonstrate event-driven communication and real-time data exchange between multiple users.

## Overview
This project focuses on implementing real-time messaging using WebSockets. It showcases how client-server communication can be handled efficiently for low-latency applications.

## Key Features
- Instant real-time messaging
- Multi-user communication support
- Event-based message handling
- Lightweight backend architecture

## Tech Stack
- Node.js
- Express.js
- Socket.IO
- JavaScript

## Skills Demonstrated
- Real-time system design
- Event-driven programming
- Backend development with Node.js
- WebSocket-based communication
